# BrainQuiz - Quiz Interativo em Java

BrainQuiz é uma aplicação de quiz inspirada no formato do Kahoot, desenvolvida em Java. Permite criar e responder perguntas, com sistema de pontuação automática e interface interativa para utilizadores.

## 🚀 Funcionalidades
- Criação de quizzes e perguntas personalizadas
- Sistema de respostas e pontuação automática
- Estrutura modular e extensível para adicionar funcionalidades futuras
- Interface simples em linha de comando (CLI)

## 🛠️ Tecnologias utilizadas
- Java
- Programação Orientada a Objetos (POO)
